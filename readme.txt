
 ★実行方法
 Windows10 + 11.0.5-BellSoftにて動作確認済み
 compile.batを実行してから、start.batを実行する
 
 
 ★道路作成

[道路]
- 配置したい道路を選択して、マスをクリックすると配置できる
- ドラッグすることで連続的に配置できる
- 道路は左上のマスを除いて、全てのマスに配置可能
- ただし、道路は一本道でないとシミュレーションは開始できない

[信号]
- 信号を選択してから、道路が配置されているマスをクリックすると信号を配置できる
- 赤時間・青時間の数値を変えてから配置すると、シミュレーション中の信号の切り替わる時間を調整できる
- 配置してある信号を左クリックすることで、選択状態になる
- 選択状態になると左下のボックスが有効になり、赤時間・青時間の編集が可能になる
- 更新ボタンを押すと赤時間・青時間の編集が反映される
- 削除ボタンを押すと選択状態の信号を削除できる
- 配置してある信号を右クリックすることで、初期状態を変更できる

[車]
- 車を選択してから、道路が配置されているマスをクリックすると車を配置できる
- 色と速度を変えてから配置すると、選択した色の車が配置され、シミュレーション中の車の速度を調整できる
- 配置してある車を左クリックすることで、選択状態になる
- 選択状態になると左下のボックスが有効になり、色と速度の編集が可能になる
- 更新ボタンを押すと色と速度の編集が反映される
- 削除ボタンを押すと選択状態の車を削除できる

「保存」ボタンで作成した配置状態をファイルに保存ができる
「ファイルを開く」ボタンで保存してあるファイルから、配置状態を復元できる
ファイルを直接ドラッグアンドドロップすることでも「ファイルを開く」と同様のことができる
「リセット」ボタンで配置状態を初期状態に戻すことができる


★動作
シミュレーション可能な道路の状態なら、スタートボタンを押すとシミュレーションが開始される
ストップボタンを押すとシミュレーションが停止する
「mutex」のチェックを外すと排他制御を行わなくなり、車の追い越しが起きるようになる
スライダーでシミュレーションのスピードを調整可能

[3Dモニター]
- シミュレーションが開始すると3Dモニターが起動する
- ←→で方向切り替え、zで前進、xで後進ができる
- z + spaceで高速移動できる
- マス上に追加された黒い点をドラッグすると直接移動できる


★tmpフォルダ
3Dモデルを車の色パターンごとに生成した一時ファイルを保存する
プログラム終了時にtmpフォルダの中をすべて削除するので注意
